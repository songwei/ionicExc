# Ionic Practice Project

## Required Installation
- node.js (8.x.x)
- git
- ionic cli: 
`npm install -g ionic`
- cordova:
`npm install -g cordova`


## Create project
    # blank template; no Ionic Appflow SDK; 
    ionic start ionicExc --type=angular
    
    # Install @ionic/lab? Yes
    ionic serve -i
